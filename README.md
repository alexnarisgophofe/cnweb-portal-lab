# cnweb-portal-lab

一个用于归档和展示多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为网页资源集合的集中存放与版本管理用途。

## 项目简介

本项目旨在提供一个轻量的 HTML 页面归档空间，方便开发者、设计师或内容创作者将零散的网页文件以结构化方式组织、存档与分享。所有页面均为独立 HTML 文件，彼此之间无强依赖关系，可单独运行或浏览。

## 目录说明

```
cnweb-portal-lab/
├── pages/            # 存放各独立 HTML 页面文件
├── assets/           # 公共资源（如图片、样式表、脚本等）
├── index.html        # 仓库入口页面（可选）
└── README.md         # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立的页面项目，命名应简洁明了。
- `assets/` 目录用于存放多个页面可能共用的资源文件，以减小重复。
- `index.html` 可作为导航页或预览页，方便快速浏览仓库内容。

## 页面归档说明

- 所有 HTML 页面应遵循标准 HTML5 规范，确保主流浏览器兼容。
- 每个页面建议附带简要说明（可在页面内注释或单独文档中记录）。
- 页面之间不应有硬编码的绝对路径依赖，推荐使用相对路径引用资源。
- 归档时请保持文件结构清晰，避免过多嵌套。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有内容。
- 提交新页面时，请在 `pages/` 下创建对应文件夹，并确保所有依赖资源已包含。
- 请勿上传包含恶意代码、侵权内容或违反平台政策的页面。
- 本仓库主要用于学习、实验与归档用途，不保证对外部链接或第三方资源的可用性。
